# Contributing to Interactive Garden

First off, thank you for considering contributing to the Interactive Garden! It's people like you that make the open-source community such an amazing place.

## How Can I Contribute?

### Reporting Bugs
If you find a visual glitch, a performance issue, or a broken link:
1. Check the [Open Issues](https://github.com/vivianchi/interactive-garden/issues) to see if it has already been reported.
2. If not, open a new issue. Include details like your browser, OS, and steps to reproduce the bug.

### Suggesting Enhancements
Have an idea for a new feature?
1. Open an issue with the "enhancement" tag.
2. Describe the feature and why it would be a good addition to the project.

### Pull Requests
Ready to contribute code?
1. Fork the repo.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Technical Details

- **Video Processing**: If you find better compression settings for the `assets/garden-interaction.mp4`, please let us know!
- **CSS Masks**: The triangular zones are defined in `styles.css`. If you improve the vertices to better match the 3D scene, pull requests are welcome.

---

Thank you for your help in making this project better!
